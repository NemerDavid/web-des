Personal Guest Link Generator

Enter one invitation group per line. Each generated link contains the invited names and the maximum guest limit. In the invitation, each person gets their own Yes/No RSVP choice.

How to use:
Write one invite group per line.
Celia, Raymond creates one link with Generate Personal Links.
On the invitation, Celia and Raymond each choose Yes or No separately.

The invitation URL is set inside the script. Existing guest, guests, id, and limit parameters are replaced automatically.
