Guest Limit Link Generator

Paste the invitation URL, then choose the limits you want. This creates shared links such as ?limit=2, ?limit=3, etc. The invitation will always display Dear Guest, and guests will type their own name in the RSVP form.

How to use:
Send the limit 2 link to all guests/families allowed 2 seats.
Send the limit 3 link to all guests/families allowed 3 seats.
No guest name is included in the link.

Each generated link contains only ?limit=2. If the base URL already has parameters, they are preserved and only limit is updated.
