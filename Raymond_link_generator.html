Common Guest Slot Link Generator

Create one reusable invitation link for each guest count. The invitation will only show boxes like Guest 1, Guest 2, Guest 3, and each guest writes their own name.

How to use:
Choose the largest invitation size you need, then generate the reusable links.
Use the 1 guest link for solo invites, the 2 guests link for couples, the 3 guests link for groups of three, and so on.
On the invitation, guests will see only Guest 1, Guest 2, etc. and write their own names.

The invitation URL is set inside the script. Existing guest, guests, invite, group, id, and limit parameters are replaced automatically. The final link only contains the guest slot count.
